## Instalasi

1. **Kloning Repositori:**
    ```
    git clone https://github.com/rahulpoonia29/To-Do-List-React
    ```
2. **Navigasi ke Direktori Proyek:**
    ```
    cd list
    ```
3. **Instal Dependensi:**
    ```
    npm install
    ```
4. **Menjalankan Program:**
    ```
    npm run dev
    ```
